# ⚡ Capital Électrique — Site Web

Site web professionnel bilingue (FR/EN) pour **Capital Électrique**, électricien licencié à Montréal.

## 🌐 Aperçu

Un site web vitrine moderne, mobile-first, conçu pour convertir les visiteurs en clients. Entièrement en HTML/CSS/JS — aucune dépendance, aucun framework requis.

## ✅ Fonctionnalités

- **Bilingue FR / EN** — bascule instantanée sans rechargement de page
- **100% Mobile-first** — responsive sur tous les appareils (téléphone, tablette, desktop)
- **Menu hamburger** — navigation fluide sur mobile avec overlay plein écran
- **Bouton d'appel sticky** — visible en permanence sur mobile pour maximiser les conversions
- **Arrière-plans photo** — images d'installation électrique réelles avec overlay sombre
- **Animations au défilement** — apparition progressive des éléments
- **SEO-ready** — balises meta incluses
- **Zéro dépendance** — un seul fichier `index.html`, aucun npm requis

## 📁 Structure

```
capital-electrique/

```

## 🚀 Mise en ligne

### Option 1 — GitHub Pages (gratuit)
1. Créer un dépôt GitHub (ex: `capital-electrique`)
2. Uploader `index.html` et `README.md`
3. Aller dans **Settings → Pages → Source → main branch**
4. Votre site est en ligne sur `https://votreusername.github.io/capital-electrique`

### Option 2 — Hébergement web
Uploader `index.html` à la racine de votre hébergeur (cPanel, FTP, etc.)

### Option 3 — Netlify (gratuit, drag & drop)
Glisser le dossier sur [netlify.com/drop](https://netlify.com/drop)
